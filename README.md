# Python 基礎教學網站

這是一個基於 Python Institute "Python Essentials 1" 課程的繁體中文教學網站。

## 特色
- **繁體中文**：所有內容皆翻譯為繁體中文，適合中文使用者。
- **互動式練習**：內建選擇題元件，即時驗證學習成果。
- **VitePress 驅動**：高效、快速且支援 Markdown。

## 本地開發

```bash
# 安裝依賴
npm install

# 啟動開發伺服器
npm run docs:dev
```

## 自動部署
本專案配置了 GitHub Actions，當推送到 `main` 分支時會自動部署到 GitHub Pages。

## 內容來源
內容源自於 EPUB 電子書 `Python Essentials 1 Beginner Course`。
