# 第 2.2 節 – Python 字面值 (Literals)

現在是時候談談 Python 的**字面值 (Literals)** 了。

## 2.2.1 字面值 – 資料本身

字面值是指其值由其本身決定的資料。

例如，數字 `123`。你知道它代表「一百二十三」。它就是一個字面值。
但如果是字母 `c`，它可能代表光速、積分常數或變數名稱。如果不看上下文，你無法確定。因此，`c` 不是字面值。

您使用字面值來編碼資料並將其放入您的程式碼中。

## 2.2.2 整數 (Integers)

Python 的整數就是不含小數部分的數字。
*   **一般寫法**：`11111111`
*   **底線寫法 (Python 3.6+)**：為了提高可讀性，您可以使用底線隔開數字，例如 `11_111_111`。Python 會忽略底線。
*   **負數**：在前面加上負號，例如 `-11111111`。

### 八進位與十六進位
*   **八進位 (Octal)**：以 `0o` 或 `0O` 開頭。數字範圍為 [0..7]。例如 `0o123` 等於十進位的 `83`。
*   **十六進位 (Hexadecimal)**：以 `0x` 或 `0X` 開頭。例如 `0x123` 等於十進位的 `291`。

## 2.2.3 浮點數 (Floats)

浮點數是用於表示含有（或可能含有）小數部分的數字。
*   **一般寫法**：`2.5`, `-0.4`
*   **簡寫**：`.4` 代表 `0.4`；`4.` 代表 `4.0`。
*   **科學記號**：使用 `e` 或 `E` 表示。
    *   `3E8` 代表 $3 \times 10^8$。
    *   `6.62607E-34` 代表普朗克常數。

> **注意**：`4` 是整數，而 `4.0` 是浮點數。雖然它們的值在數學上相等，但在 Python 中它們的資料型別不同。

## 2.2.4 字串 (Strings)

字串用於處理文本。字串必須被包含在引號中。
*   **引號**：可以使用單引號 `'` 或雙引號 `"`。
*   **跳脫字元**：如果您想在字串中包含引號，可以使用反斜線 `\`。
    *   `"I like \"Monty Python\""`

## 2.2.5 布林值 (Boolean Values)

布林值只有兩個：
*   **True** (真)
*   **False** (假)

它們來自於邏輯學中的真偽判斷。注意首字母必須大寫。
