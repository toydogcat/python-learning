# 下一步：PCEP 認證與進階學習

恭喜您完成了 **Python Essentials 1** 課程！您現在已經掌握了 Python 程式設計的基礎知識。

## PCEP 認證考試

本課程的內容涵蓋了 **PCEP (Python Certified Entry-Level Python Programmer)** 認證考試的大部分範圍。

### 考試大綱摘要

1.  **電腦程式設計與 Python 基礎 (18%)**
    *   理解直譯、編譯、語法與語義。
    *   字面量、變數、命名規範、進位系統。
2.  **流程控制 (29%)**
    *   條件語句 (`if`, `if-else`, `if-elif-else`)。
    *   迴圈 (`while`, `for`) 與控制 (`break`, `continue`)。
3.  **資料集合 (25%)**
    *   列表 (Lists) 的操作與切片。
    *   元組 (Tuples) 的不可變性。
    *   字典 (Dictionaries) 的鍵值對操作。
4.  **函式與例外處理 (28%)**
    *   函式定義、參數傳遞、返回值、作用域。
    *   例外處理 (`try-except`) 機制。

## 接下來您可以做什麼？

1.  **實作專案**：嘗試撰寫一些小工具，如計算機、簡單的爬蟲或自動化腳本。
2.  **進階課程**：繼續學習 **Python Essentials 2**，深入了解模組、套件、字串處理、物件導向程式設計 (OOP) 以及檔案處理。
3.  **參加認證**：前往 [Python Institute 官網](https://pythoninstitute.org) 獲取認證，這對您的職涯發展將大有幫助。

祝您在 Python 的學習旅程中一帆風順！
