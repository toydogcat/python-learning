# 關於本課程

歡迎來到 **Python Essentials 1**！本課程由 [OpenEDG Python Institute](https://pythoninstitute.org) 與 Cisco Networking Academy 合作設計開發。

Python 是世界上成長最快的程式語言之一，幾乎應用於從遊戲、醫學到核子物理的每一個領域。對於任何未來的程式設計師來說，擁有 Python 的基礎知識是至關重要的。

## 課程目標

本課程旨在教您 Python 程式設計的基礎知識，即使您完全沒有程式設計經驗。此外，它還能幫助您準備參加 **PCEP (Python Certified Entry-Level Python Programmer)** 認證考試。
