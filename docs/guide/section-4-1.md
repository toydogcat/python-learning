# 第 4.1 節 – 函式 (Functions)

歡迎來到第四模組！在本節中，您將學習如何建立、使用和呼叫您自己的函式。

## 4.1.1 為什麼需要函式？

在編寫程式時，您可能會發現某段程式碼被重複使用了多次。如果您直接複製貼上這些程式碼，一旦發現錯誤，您就必須修改所有出現過的地方，這既費時又容易出錯。

**函式的好處：**
1.  **程式碼重用**：寫一次，到處呼叫。
2.  **分解問題 (Decomposition)**：將大型複雜的問題分解成多個小的、易於管理和測試的部分。
3.  **易於維護**：修改函式內部的邏輯，所有呼叫該函式的地方都會同步更新。

## 4.1.2 函式的來源
1.  **內建函式 (Built-in functions)**：如 `print()`, `input()`, `len()`，由 Python 原生提供。
2.  **模組函式 (Preinstalled modules)**：來自 Python 標準庫的模組，需要 `import` 後使用。
3.  **自定義函式 (User-defined functions)**：由您自己在程式碼中編寫的函式。

## 4.1.3 定義您的第一個函式

使用關鍵字 **`def`** 來定義函式：

```python
def message():
    print("請輸入一個值：")

# 定義完後，必須呼叫它才會執行
print("程式開始")
message()  # 函式呼叫
print("程式結束")
```

**函式定義的結構：**
*   `def` 關鍵字開頭。
*   函式名稱（遵循變數命名規則）。
*   一對括號 `()`。
*   結尾的冒號 `:`。
*   **縮排**的程式碼區塊（函式體）。
