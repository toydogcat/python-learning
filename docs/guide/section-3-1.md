# 第 3.1 節 – 在 Python 中做決定

歡迎來到第三模組！在第一節中，我們將學習條件語句以及如何使用它們在 Python 中做出決定。

## 3.1.1 問題與回答

程式設計師編寫程式，程式提出問題。電腦執行程式並提供答案。程式必須能夠根據收到的答案做出反應。

幸運的是，電腦只知道兩種答案：
*   **是**，這是真的 (True)。
*   **否**，這是假的 (False)。

為了提出問題，Python 使用了一組特殊的運算子。

## 3.1.2 比較運算子

### 相等運算子 (`==`)
問題：兩個值是否相等？
*   `=` 是**賦值**（將右邊的值給左邊）。
*   `==` 是**比較**（檢查兩邊是否相等）。

### 不等運算子 (`!=`)
問題：兩個值是否不相等？

### 大於、小於系列
*   `>` (大於)
*   `>=` (大於或等於)
*   `<` (小於)
*   `<=` (小於或等於)

## 3.1.3 條件語句 (if, if-else, if-elif-else)

### if 語句
如果條件成立，就執行縮排的程式碼。
```python
if sheep_counter >= 120:
    sleep_and_dream()
```

### if-else 語句
提供「方案 B」。如果 if 條件不成立，就執行 else 部分。
```python
if the_weather_is_good:
    go_for_a_walk()
else:
    go_to_a_theater()
```

### if-elif-else 瀑布結構
當有多個條件需要依序檢查時使用。
```python
if x > 0:
    print("正數")
elif x < 0:
    print("負數")
else:
    print("零")
```

## 3.1.4 縮排 (Indentation) 的重要性
在 Python 中，縮排不僅是為了好看，它是**語法的一部分**。縮排定義了哪些程式碼屬於哪個區塊。建議使用 **4 個空格**進行縮排。
