# 第 2.3 節 – 運算子：資料處理工具

在本節中，我們將討論 Python 的運算子。

## 2.3.1 Python 作為計算機

你可以直接在 `print()` 函式中進行數學運算。
```python
print(2 + 2)
```
這會輸出 `4`。雖然這不是最方便的計算機，但它展示了 Python 處理**運算子 (Operators)** 和**運算式 (Expressions)** 的能力。

## 2.3.2 基礎運算子

Python 支援以下基礎算術運算子：
*   `+` (加)
*   `-` (減)
*   `*` (乘)
*   `/` (除)
*   `//` (整數除法 / 地板除法)
*   `%` (取餘數 / 模數)
*   `**` (指數 / 乘方)

### 指數運算 (`**`)
```python
print(2 ** 3)   # 8
print(2 ** 3.)  # 8.0
```
**規則**：如果兩個參數都是整數，結果是整數；如果至少有一個是浮點數，結果是浮點數。

### 乘法 (`*`)
同樣遵循「整數 vs. 浮點數」規則。

### 除法 (`/`)
**注意**：除法運算子的結果**永遠是浮點數**，即使結果看起來像整數（例如 `6 / 3` 的結果是 `2.0`）。

### 整數除法 (`//`)
也稱為地板除法 (Floor division)。它會將結果無條件捨去到最接近的較小整數。
```python
print(6 // 4)   # 1
print(-6 // 4)  # -2 (因為 -2 比 -1.5 小)
```

### 取餘數 (`%`)
回傳整數除法後的餘數。
```python
print(14 % 4)   # 2 (14 // 4 = 3, 餘數 2)
```

> **重要**：絕對不要嘗試**除以零**，這會導致錯誤。

## 2.3.3 運算子優先級與結合性

當一個運算式中有多個運算子時，Python 會根據優先級來決定執行順序：

1.  `**` (最高)
2.  正負號 (`+`, `-` 單元運算子)
3.  `*`, `/`, `//`, `%`
4.  `+`, `-` (二元運算子) (最低)

### 結合性 (Binding)
大多數運算子是**左結合**（由左至右運算）。
例外：**指數運算子 (`**`) 是右結合**。
```python
print(2 ** 2 ** 3)  # 等於 2 ** (2 ** 3) = 2 ** 8 = 256
```
